# Spark

## Download Spark

Download and unzip [latest Spark release](https://spark.apache.org/downloads.html):

```shell
cd ~/Downloads
pwd
curl -L -o spark.tgz https://mirror.netcologne.de/apache.org/spark/spark-3.0.1/spark-3.0.1-bin-hadoop3.2.tgz
```
